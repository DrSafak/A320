7.0 Flight Controls

Flight Control Displays

F/CTL Page Description 2

1. HYDRAULIC SYS PRESSURE INDICATION - Displays green when pressurization is normal; changes to amber when low pressure detected.

2. ELEVATOR ACTUATOR INDICATION - Displays green; changes to amber when the associated actuator is not available. Partially boxed amber if associated computer fails.

3. RUDDER TRIM POSITION - Displays blue; changes to amber if rudder trim reset fails. Amber Xs appear if the aircraft reverts to the yaw mechanical mode.
