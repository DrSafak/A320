5.0 Engines and APU

Engine Start and Ignition Panel

Engine Mode Selector

CRANK - The start valve opens if the ENG MAN START switch is ON and N2 is less than 10%. Ignition is not supplied.

NORM - When the NORM position is selected, continuous ignition A and B is automatically selected when an engine is running and one of the following is met:

  • FLX or TO/GA thrust is selected on the ground
  • TO/GA thrust is selected in flight
  • ENG ANTI ICE switch is ON
  • Engine surge or stall occurs in flight
  • FLAPS lever position is other than 0 in flight (approach idle is selected)

IGN/START IN FLIGHT - When the ENG MODE selector is positioned to IGN/START inflight the ENG MASTER switch is armed and provides continuous ignition using both igniters.
